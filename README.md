# Shared Shopping Dashboard

Ein interaktives React-Dashboard für gemeinsames Einkaufs- und Bestandsmanagement in Haushalten, WGs oder Teams.

## Features

✨ **Gemeinsame Einkaufsliste** - Mehrere Personen können parallel Einkäufe planen und abarbeiten
💰 **Preisübersicht** - Verfolgung der Kosten pro Eintrag und Person
📦 **Bestandsverwaltung** - Inventarverfolgung mit Mindestbeständen und Status-Alarmen
📸 **Foto-Upload** - Belege, Kassenzettel und Inventarfotos speichern
📊 **KPIs** - Live-Metriken zu Inventarwert, Erledigungsquoten und Kostenverteilung

## Technologie Stack

- **React 18** via CDN (keine Build-Tools nötig!)
- **Babel Standalone** für JSX-Rendering
- **Vanilla CSS** mit CSS-Variablen
- **LocalStorage-Ready** für Weitergabe an Backend

## Verwendung

1. Öffne `index.html` im Browser
2. Wechsle durch die Tabs: Dashboard → Liste → Bestand → Uploads
3. Füge Produkte hinzu, verwalte Bestände und plant Einkäufe gemeinsam!
